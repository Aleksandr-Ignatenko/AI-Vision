# 🖤 AI Vision — Admin / Roadmap Plan

## 🎯 Цель проекта
Создать Telegram-бота для генерации **платных изображений, видео, музыки и песен** с помощью ИИ-нейросетей.

На первом этапе используется **только бесплатная инфраструктура без ограничений по времени**,  
с последующим переходом на платные тарифы при росте нагрузки и начале полноценного продакшена.
